# 🏫 座席替えツール v3.1.1 

学校の先生が、「男女交互」「特定の生徒の前方配置」「NGペア回避」などの複雑な条件を数秒で解決するためのツールです。

## 🌐 ツールを使う（インストール不要）

以下のURLにアクセスするだけで、今すぐブラウザでご利用いただけます。

# 👉 [座席替えツールを起動する](https://uhhh-noname.github.io/sekigae/)

※データはサーバーに送信されず、あなたのブラウザ内だけで処理されるため安全です。

---

### 💡 初めてご利用になる方へ（おすすめの導入手順）

本ツールは過去の履歴を学習します。初回は以下の手順で**「今の座席」**を登録することをおすすめします。

1. **テンプレートの入手** まずツールを起動し、画面上部の **「① テンプレートDL」** ボタンからCSVファイルを保存してください。
2. **ベース名簿の作成** 保存したCSVをExcel等で開き、出席番号・氏名・性別を入力します。**この時点では座席順はバラバラで全く問題ありません。**
3. **名簿の読み込みと再現** 作成したCSVをツールで読み込み、一度座席を生成します。その後、手動入れ替え機能（後述）を使い、**実際の教室の座席と全く同じ配置**になるよう画面上で整えます。
4. **初期データとして保存** 配置が整ったら、そのまま**「③ CSV保存」**をクリックして保存します。
5. **次回以降の運用** 次回の座席替えからは、手順4で保存したCSVを読み込んでください。「前回と同じ席やペアを避ける」機能がフル活用されるようになります。

---

## ⚙️ 設定項目の説明（パラメータ）

生成前に調整できる各項目の効果は以下の通りです。

| 項目 | 設定内容と効果 |
| :--- | :--- |
| **列数** | 教室の横列の数を指定します（例：6列など）。 |
| **並び替えモード** | **男・女 / 女・男**: 横並びのペアを男女（または女男）で固定します。<br>**男女交互 / 女男交互**: 教室全体を市松模様状に交互配置します。<br>**ランダム**: 性別を考慮せず、自由に配置します。 |
| **前方優先行数** | CSVで「1」をつけた生徒を、前から何行目までに収めるかを指定します。 |
| **試行回数** | 内部での計算回数です。回数を増やすほど「NGペア回避」や「履歴」をより厳密に考慮します。 |

## 🚀 詳しい使い方

### 1. 準備（テンプレートの作成）
* 画面の **「① テンプレートDL」** ボタンから、名簿管理用のCSVファイルを保存します。
* Excelなどで開き、名簿を入力します（出席番号、氏名、性別、NG相手など）。

### 2. 名簿の読み込みと「前方優先」の調整
* **「ファイルを選択」** からCSVファイルを読み込みます。
* 画面左側の **「【前回】の座席表」** の生徒をクリックすると、その場で「前方優先（赤タグ）」のオン/オフを切り替えられます。

### 3. 条件設定と生成
* 列数や並び順を選択し、 **「② 座席を生成」** をクリックします。
* 数万回の計算から、条件に最も適合する配置が右側に表示されます。

### 4. 手動での「位置移動（入れ替え）」
* 生成された座席を微調整したい場合は、以下の手順で自由に入れ替えが可能です。
    1. 入れ替えたい **1人目をクリック**（黄色く光ります）。
    2. 移動先の **2人目（または空席）をクリック**。
    3. 二人の位置が入れ替わります。

### 5. 保存と印刷
* 結果が良ければ **「③ CSV保存」** をクリックします。
* **印刷**: ブラウザの印刷機能（Ctrl + P）を使うと、掲示用の座席表のみが綺麗にレイアウトされます。

## 🔒 プライバシーとセキュリティ
本ツールは**「完全ローカル処理」**です。名簿データや座席結果がインターネット上のサーバーに送信されることは一切ありません。校務用PCでも安心してご利用いただけます。
