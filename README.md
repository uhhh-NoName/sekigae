# 🏫 座席替えツール v3.1.1 

学校の先生が、「男女交互」「特定の生徒の前方配置」「NGペア回避」などの複雑な条件を数秒で解決するためのツールです。

## 🌐 ツールを使う（インストール不要）

以下のURLにアクセスするだけで、今すぐブラウザでご利用いただけます。  
# 👉 [https://uhhh-noname.github.io/sekigae/](https://uhhh-noname.github.io/sekigae/)  
※データはサーバーに送信されず、あなたのブラウザ内だけで処理されるため安全です。

---

## 📥 自分のPCに保存して使う方法
ネット環境がない場所でも、ファイルをダウンロードしておけば利用可能です。

1. 右上の緑色のボタン **「Code」** をクリックします。
2. **「Download ZIP」** を選択します。
3. ダウンロードされたZIPファイルを展開（右クリックして「すべて展開」）します。
4. 中にある **`index.html`** をダブルクリックすると、ブラウザでツールが起動します。

---

## 🚀 詳しい使い方

### 1. 準備（テンプレートの作成）
* 画面の **「① テンプレートDL」** ボタンから、名簿管理用のCSVファイルを保存します。
* Excelなどで開き、名簿を入力します（出席番号、氏名、性別など）。
* **前方優先**: 「1」を入れると、その生徒は必ず前方の席になります。
* **NG相手**: 出席番号を入れると、その二人は隣り合わせになりません。

### 2. 名簿の読み込みと生成
* **「ファイルを選択」** から、作成したCSVファイルを読み込みます。
* **列数・並び順**（男女交互、ペア、ランダムなど）を選択します。
* **「② 座席を生成」** をクリックすると、最適な配置を表示します。

### 3. 保存と次回への引き継ぎ
* 結果が良ければ **「③ CSV保存」** をクリックして保存します。
* **ポイント**: 保存したCSVには「今回のペア履歴」が記録されます。次回の座席替えでそのファイルを読み込むと、**「連続で同じ隣の席」にならないよう自動で配慮**されます。

### 4. 印刷
* ブラウザの印刷機能（Ctrl + P）を使うと、掲示用の座席表のみが綺麗にレイアウトされます。

## 🔒 プライバシーとセキュリティ
本ツールは**「完全ローカル処理」**です。名簿データや座席結果がインターネット上のサーバーに送信されることは一切ありません。校務用PCでも安心してご利用いただけます。
