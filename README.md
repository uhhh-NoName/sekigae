# 🏫 座席替えツール v3.1.1 

学校の先生が、**「男女交互」「特定の生徒の前方配置」「NGペア回避」**などの複雑な条件を数秒で解決するためのツールです。
ブラウザだけで動作し、データは外部に送信されないため、個人情報の保護も万全です。

---

## 🚀 使い方ステップ

### 1. 準備（テンプレートのダウンロード）
* **「① テンプレートDL」**ボタンを押し、名簿管理用のCSVファイルをダウンロードします。
* Excelなどで開き、名簿を入力します（出席番号、氏名、性別など）。
* **「前方優先」**に「1」を入れると、その生徒は必ず前方の席になります。
* **「NG相手」**に出席番号を入れると、その二人は隣り合わせになりません。

### 2. 名簿の読み込み
* **「ファイルを選択」**から、作成したCSVファイルを読み込みます。
* 読み込まれると、画面左側に**「前回の座席」**が表示されます。
    * ※前回座席の生徒をクリックすると、その場で「前方優先（赤タグ）」の切り替えが可能です。

### 3. 条件の設定と生成
* **列数・並び順**（男女交互、ペア、ランダムなど）を選択します。
* **「② 座席を生成」**をクリックすると、数万回のシミュレーションから最適な配置を表示します。

### 4. 手動での微調整
* 生成された座席表の生徒をクリックすると黄色く光ります。
* 別の生徒（または空席）をクリックすると、**二人を入れ替える**ことができます。

### 5. 保存と印刷
* 結果が良ければ**「③ CSV保存」**をクリックします。
    * ※このCSVには「今回のペア履歴」が含まれます。次回このファイルを読み込むと、**「連続で同じ隣の席」にならないよう自動配慮**されます。
* ブラウザの印刷機能（Ctrl + P）を使うと、設定パネルが隠れ、**掲示用の座席表のみが綺麗に印刷**されます。

---

## ✨ 主な機能
* **市松模様配置**: 男女を完全交互に並べます。
* **前方優先対応**: 視力や配慮が必要な生徒を、指定した行数より前に配置します。
* **履歴学習**: 過去に隣になった生徒を記録し、極力ペアが重複しないようにします。
* **レスポンシブ対応**: PCでもタブレットでも操作しやすい画面設計です。

---

## 🔒 プライバシーについて
入力された名簿や座席データは、あなたのブラウザ内でのみ処理されます。**インターネット上のサーバーには一切送信されません**ので、安心して校務にご利用ください。
